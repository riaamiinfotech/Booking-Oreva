Create flexible quantity-based bulk discount rules for WooCommerce products and categories.

== Description ==

Rise Bulk Discounts for WooCommerce allows store owners to create powerful and flexible bulk pricing rules based on product quantity.

You can define minimum and maximum quantity ranges, apply percentage discounts, and control rules at the category level.  
The plugin is lightweight, easy to configure, and works seamlessly with WooCommerce without modifying core files.

### Key Features

* Create unlimited bulk discount rules
* Set minimum and maximum quantity ranges
* Apply percentage-based discounts
* Enable or disable rules instantly
* Assign rules to product categories
* Simple and clean admin interface
* Compatible with latest WooCommerce versions

== Installation ==

1. Upload the `rise-bulk-discounts-for-woocommerce` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the **Plugins** menu in WordPress
3. Go to **Rise Bulk Discounts** in the WordPress admin menu
4. Create your first bulk discount rule

== How It Works ==

1. Add a new discount rule
2. Select a parent product category
3. Set minimum and maximum quantities
4. Define the discount percentage
5. Enable the rule and save

The discount is automatically applied in the cart when conditions are met.

== Frequently Asked Questions ==

= Does this plugin work with variable products? =
Yes, bulk discounts apply based on the total quantity in the cart.

= Can I disable a rule temporarily? =
Yes, each rule has an enable/disable toggle.

= Does it modify WooCommerce core files? =
No, the plugin uses standard WooCommerce hooks and filters.
